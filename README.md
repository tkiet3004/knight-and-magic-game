# Knight And Magic

**Genre**: 2D Platformer
**Setting**: Medieval Fantasy

## Project Introduction
A 2D platformer game featuring co-op gameplay that combines puzzle-solving and combat. Set in a Medieval world, two players control characters with distinct abilities: a **Mage** and a **Knight**. They must coordinate to overcome obstacles, solve puzzles, and fight enemies in challenging environments. The main goal is to test the players' coordination and creativity.

## Concepts
Players take on the role of companions to solve puzzles based on the coordination between physical strength and magic.
- **Knight**: Can push heavy objects and fight enemies with physical attacks.
- **Mage**: Uses magic to support combat and interact with magical elements.

## Technical Analysis
- **Engine**: Unity (2D)
- **Assets**: Unity Asset Store, OpenGameArt, Itch.io
- **Art Style**: 2D / Pixel Art, minimalistic.

## Audio
- **Sound FX**: Categorized by role (Special Effects, Weapon fire, Explosions).
- **Music**: Theme-based (Shell Screen, Level Theme).

## Current Focus
- **Phase 1**: Knight's Journey (Single player mechanics for the Knight first).
- **Phase 2**: Co-op mechanics and Mage implementation.
